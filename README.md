# DevLaunch – Responsive SaaS Landing Page

DevLaunch is a responsive marketing landing page built using semantic HTML5 and modern CSS. The project simulates a real-world SaaS/EdTech product website with structured layout and clean design.

## 🚀 Project Overview

This project includes:

- Hero section with call-to-action
- Features section with card layout
- Pricing plans comparison
- Responsive footer
- Dropdown navigation
- Fully responsive layout using Flexbox

The goal of this project was to practice:

- Semantic HTML structure
- Flexbox layout systems
- Responsive design techniques
- Component-based CSS structure
- Clean spacing and visual hierarchy

## 🛠 Technologies Used

- HTML5
- CSS3
- Flexbox
- Google Fonts (Inter)

## 📱 Responsive Design

The layout adapts to different screen sizes:
- Desktop layout with horizontal sections
- Tablet and mobile layout with stacked components
- Adjusted typography and spacing for small screens

## 🎯 Learning Focus

This project was built without using tutorials to strengthen independent problem-solving skills and layout understanding.
